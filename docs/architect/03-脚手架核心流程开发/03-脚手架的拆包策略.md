# 脚手架拆包策略

## 拆包结果

- 核心流程：core

- 命令：commands

  - 初始化

  - 发布

  - 清除缓存

- 模型层：models

  - Command 命令

  - Project 项目

  - Component 组件

  - Npm 模块

  - Git 仓库

- 支撑模块：utils

  - Git 操作

  - 云构建

  - 工具方法

  - API 请求

  - Git API

## 拆分原则

根据模块的功能拆分：

- 核心模块：core

- 命令模块：commands

- 模型模块：models

- 工具模块：utils
