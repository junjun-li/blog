# 初始Java

```java
// HelloWord这个叫标识符 类的名字
// 不能以数字开头
// 严格大小写
// 不能使用关键字
// 标识符的命名最好能反应其作用
public class HelloWord {
    public static void main(String[] args) {
        System.out.println("hello word");
        System.out.print("hello word"); // 输出之后不换行
        System.out.println("hello word");
    }
}

```
