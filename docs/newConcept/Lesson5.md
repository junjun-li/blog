## Miss,Mrs,Ms

Miss 小姐(一般指未婚女性)

Mrs 太太(已婚女性)

Ms 女士(不确定结婚没)

## 时间单词

- morning 早晨

```
in the morning 在早上
Good morning! 早上好
```

- afternoon 下午

```
in the afternoon 下午好
Good afternoon. 中午好
```

- evening 晚上

```
in the evening 在晚上
Good evening 晚上好
```

- night 夜里(这里的用法和上面有所区分)

```
at night 在夜里
Good night. 晚安
```
