## 注意事项

1. 避免浮躁, 按部就班

算法是枯燥的, 需要静下心来好好沉淀

2. 不投机取巧, 循序渐进

不要做了一道题, 就想着所有的算法都会了, 不要急于求成

3. 难度大的算法题目如何解

算法的本质是寻找规律并实现, 发现`输入`和`输出`的关系, 寻找突破点

找规律这种事, 只能自己悟, 悟不出来就凉凉, 问别人也是问不会的, 静下心来, 好好悟 🤔

实现是程序+数据结构的结合体
