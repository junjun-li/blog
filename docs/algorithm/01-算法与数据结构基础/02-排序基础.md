## 选择排序法

### SelectionSort.java

```java
public class SelectionSort {
  private SelectionSort() {}
  // <E extends Comparable<E>>
  // 对这个泛型做一个约束，通过这个约束，使这个泛型是可以比较的
  // E 这个类型，需要实现 Comparable 接口
  // Comparable 实现了这个接口，说明这个类是可以比较的
  public static <E extends Comparable<E>> void sort(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
      // 1. 假设以i为最小值
      int minIndex = i;
      for (int j = i + 1; j < arr.length; j++) {
        // 2. 判断 arr[i, n]
        // if (arr[j] < arr[minIndex]) {
        //   minIndex = j;
        // }

        // compareTo返回int，如果
        // 等于-1，表示 左 < 右
        // 等于 0，表示 左 = 右
        // 等于 1，表示 左 > 右
        if (arr[j].compareTo(arr[minIndex]) < 0) {
          minIndex = j;
        }
      }
      swap(arr, i, minIndex);
    }
  }
  private static <E> void swap(E[] arr, int i, int minIndex) {
    E temp = arr[minIndex];
    arr[minIndex] = arr[i];
    arr[i] = temp;
  }
  public static void main(String[] args) {
    Integer[] arr = {6, 4, 8, 2, 3, 1, 5, 7};
    SelectionSort.sort(arr);
    // for (int item : arr) {
    //   System.out.print(item + "");
    // }
    Student[] students = {
      new Student(10, "张三"),
      new Student(2, "李四"),
      new Student(20, "王五"),
      new Student(8, "赵六"),
      new Student(15, "田七")
    };
    SelectionSort.sort(students);
    for (Student student : students) {
      System.out.println(student);
    }
  }
}
```

### Student.java 自定义类排序

```java
public class Student implements Comparable<Student> {
  private String name;
  private int id;
  public Student() {}
  public Student(int id, String name) {
    this.name = name;
    this.id = id;
  }
  @Override
  public boolean equals(Object student) {
    // 增加判断,防止强制转换的时候报错
    if (this == student) {
      return true;
    }
    if (student == null) {
      return false;
    }
    if (this.getClass() != student.getClass()) {
      return false;
    }
    Student s = (Student) (student);
    return this.name.equals(s.name);
  }
  @Override
  public int compareTo(Student student) {
    return this.id - student.id;
  }
  @Override
  public String toString() {
    // return String.format("Student(id: %a,name: %s)", id, name);
    return "Student{" + "name='" + name + '\'' + ", id=" + id + '}';
  }
}
```

- 因为数据，已经有了一定的规模，我们不可能打印出来，肉眼判断排序是否正确，我们需要一个函数来帮我们确定排序是否正确

- 创建`SortingHelper`，来辅助我们

```java
public class SortingHelper {
  private SortingHelper() {}

  // <E extends Comparable<E>>这样子表示:
  // 传递过来的类, 必须实现 Comparable 方法
  // {1,2,3,4,5}
  // 判断一个数组, 是否是有序的
  public static <E extends Comparable<E>> Boolean isSorting(E[] arr) {
    for (int i = 1; i < arr.length; i++) {
      // 因为要比较 arr[i-1]和arr[i]的大小
      // 所以i的初始值为1

      // 等于-1，表示 左 < 右
      // 等于 0，表示 左 = 右
      // 等于 1，表示 左 > 右
      // 1.compareTo(2) > 0, 说明这个数组是无序的(没有拍好序)
      if (arr[i - 1].compareTo(arr[i]) > 0) {
        return false;
      }
    }
    return true;
  }

  // 因为后续需要比较多个算法，封装一个Test类
  public static <E extends Comparable<E>> void sortTest(String sortName, E[] arr) {
    long startTime = System.nanoTime();
    if (sortName.equals("SelectionSort")) {
      SelectionSort.sort(arr);
    }
    long endTime = System.nanoTime();
    double time = (endTime - startTime) / 1000000000.0;
    if (!SortingHelper.isSorting(arr)) {
      throw new RuntimeException(sortName + "failed");
    }
    System.out.println(String.format("%s , n = %d : %f s", sortName, arr.length, time));
  }
}
```

```java
public class SelectionSort {
  private SelectionSort() {}
  public static <E extends Comparable<E>> void sort(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
      int minIndex = i;
      for (int j = i + 1; j < arr.length; j++) {
        if (arr[j].compareTo(arr[minIndex]) < 0) {
          minIndex = j;
        }
      }
      swap(arr, i, minIndex);
    }
  }
  private static <E> void swap(E[] arr, int i, int minIndex) {
    E temp = arr[minIndex];
    arr[minIndex] = arr[i];
    arr[i] = temp;
  }
  public static void main(String[] args) {
    int[] dataSize = {10000, 100000};
    for(int n: dataSize) {
      Integer[] arr = ArrayGenerate.generateRandomArray(n, n);

}

// 数据规模差了10倍
// 说明了这个时间复杂度是 O(n^2)
```

## 插入排序法

### 基础

```java
import java.util.Arrays;
public class InsertionSort {
  public static <E extends Comparable<E>> void sort(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
      for (int j = i; j - 1 >= 0; j--) {
        if (arr[j].compareTo(arr[j - 1]) < 0) {
          swap(arr, j, j - 1);
        }
      }
    }
  }
  public static <E extends Comparable<E>> void swap(E[] arr, int i, int j) {
    E temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
  }
  public static void main(String[] args) {
    Integer[] arr = {4, 5, 2, 1, 3};
    InsertionSort.sort(arr);
    for (int item : arr) {
      System.out.println(item);
    }
  }
}

```

### 优化(使用赋值 而不是用交换)

```java
import java.util.Arrays;
public class InsertionSort {
  public static <E extends Comparable<E>> void sort(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
      for (int j = i; j - 1 >= 0; j--) {
        if (arr[j].compareTo(arr[j - 1]) < 0) {
          swap(arr, j, j - 1);
        }
      }
    }
  }
  public static <E extends Comparable<E>> void swap(E[] arr, int i, int j) {
    E temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
  }
  // 插入排序法的优化(减少寻址过程)
  public static <E extends Comparable<E>> void sort2(E[] arr) {
    for (int i = 0; i < arr.length; i++) {
      E temp = arr[i];
      int j;
      // 先保存arr[i], 找到arr[i]需要放到的位置在插入
      for (j = i; j - 1 >= 0 && temp.compareTo(arr[j - 1]) < 0; j--) {
        arr[j] = arr[j - 1];
      }
      arr[j] = temp;
    }
  }

  public static void main(String[] args) {
    Integer[] arr = {4, 5, 2, 1, 3};
    InsertionSort.sort2(arr);
    for (int item : arr) {
      System.out.println(item);
    }
  }
}
```

## 插入排序法的特征

- 插入排序法

插入排序法存在内层循环，不一定会循环完全的机制

在极端情况下，一个数组本身就是有序的，插入排序法只需要外层循环(内层循环只执行了常数级别的操作即判断)即可走完，复杂度变为 O(n)

- 选择排序法

选择排序不一样，总要扫描一次 arr[i...n) 中未排序的数字，查找其中最小的元素，所以选择排序算法的复杂度永远是 O(n^2)

## 选择排序法和插入排序法的区别

选择排序法处理整个数据中，最小的依次排好

插入排序法只处理当前元素，把处理过的元素，排序好，插入排序法永远不会动 i 还没有遍历到的元素
